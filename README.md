# Melodiary
A platform that consolidates your music library and helps you to manage, migrate and share it with friends. Work in progress!

## Features (planned)
- Support for multiple streaming services
- Unified library view across all platforms
- Manual track entry for physical media (CD, Vinyl)
- Playlist migration between services
- New release notifications from artists
- Social features

## Architecture

- Frontend: React (CloudFlare pages)
- Backend: AWS Lambda (Python)
- Database: DynamoDB
- Storage: S3
